# User filter


## Setup:
> ```shell
> # SSH
> git clone 
> # HTTPS
> git clone 
> ```


## Settings in .env file:
> `MONGODB_NAME` - Database name
> 
> `MONGODB_COLLECTION` - Collection name
> 
> `MONGODB_URL` - Database url


## How to run:
> ```shell
> # Run
> docker-compose -f docker-compose.yml up --build
> # Stop
> docker-compose -f docker-compose.yml stop
> ```
